트리거 - 부모 테이블이 업데이트 e.g) id가 업데이트 되면 자식 테이블들의 id도 업데이트가 됨

시퀀스 - create sequence	increment by 	start with		 maxvalue 자동으로 번호 생성 가능

join - 두개의 테이블을 하나처럼 연결해서 출력

nvl(value1, value2) - value1값이 null 이 아닐 경우 그대로 쓰고 null 이면 value2 값을 사용



회원 정보
회원 ID / 비밀번호 / 이름 / 나이 / 성별

영화 정보
영화 ID / 영화명 / 영화 등급 / 시간(1,2,3,4시간) / 가격

관 정보
관 ID / 관 이름(1관,2관,3관, 4D관)

리뷰 정보
리뷰 식별 ID / 회원 ID/ 글 제목 / 글 내용 / 작성일


회원정보+영화 정보
영화 예매
회원 ID / 영화 ID / 상영 시간 / 상영관 / 티켓수 / 총합 가격

영화 정보+관 정보
영화 ID / 관 ID / 날짜 / 시간

회원정보+리뷰정보
회원 ID / 리뷰 ID / 리뷰갯수에 따른 혜택
